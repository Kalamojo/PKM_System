# While Loop

>[!note]
>Loops complete until condition of while are met.

````ad-example
```cpp
#include <iostream>

int main() {
  int start = 1;

  while (start < 200) {
	  std::cout << start << std::endl;
	  start *= 2;
  }
}
````
