# Binary

```ad-note
0s or 1s. These are the only two options in the binary system. 
```

> [!info]
> For an 4 bit number:
> 
> $$d_{3}*2^{3} + d_{2}*2^{2} + d_{1}*2^{1} + d_{0}*2^{0} = \text{value}$$
> With each value being multiplied by these powers

> [!example]
> $${0010 \enspace 0110}_2 = 38_{10}$$
