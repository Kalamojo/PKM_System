# Exponential Density Function


> $$f_{X}(x \mid \lambda)=\left\{\begin{array}{cc}\lambda e^{-\lambda x} & \text { for } x>0 \\0 & \text { for } x \leq 0\end{array}\right.$$
> $\lambda = \text{rate parameter} = \frac{1}{\text{mean}}$
