# Object

> [!note]
> A C++ object is an instance of a class.

> [!example]
> ```cpp
> class school {
> 	public:
> 		int students;
> 		double size;
> }
> 
> int main() {
> 	school msa; // msa is an object
> 	return 0;
> }
> ```
