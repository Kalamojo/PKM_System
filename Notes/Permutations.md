# Permutations


> [!note]
The ways in which a set of things can be ordered or arranged.

> [!math]
> Formula for the number of permutations of length $k$ from a list of $n$ elements without repetition:
> $$\frac{n!}{(n-k)!}$$

> [!example] 
> 2,4,3,1 is a permutation of {1,2,3,4}
