# Sets


> [!note]
> A collection of objects called elements. Order and multiplicity of values do not matter.

> [!example] 
> $\{1,2,3\} = \{3,1,2\} = \{1,1,3,2,1,2\}$

>[!info]
> $x\epsilon S$ shows that $x$ is an element of set $S$
