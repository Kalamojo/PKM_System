# Domain

> [!note]
> The domain of a variable is all possible values it can be. For a [[Function]], it is the set of all [[Feature|Inputs]] it accepts.

> [!math]
> Represented by:
> $$\mathcal{X}$$

> [!example]
> - The domain of x is real numbers
> - The domain of y is foods
