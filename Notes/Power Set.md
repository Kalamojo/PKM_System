# Power Set

> [!note]
> A power set of  $S$ contains all possible subsets of $S$.

> [!example] 
> - Power set of $\{1,2\} = \{\emptyset,\{1\},\{2\},\{1,2\}\}$
> - Power set of $\{1,2,3\} =$ $\{\emptyset,\{1\},\{2\},\{3\},\{1,2\},\{2,3\},\{3,1\},\{1,2,3\}\}$
> - Power set of $\emptyset = \{\emptyset\}$
