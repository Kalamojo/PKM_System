# Bernoulli Mass Function

> [!math]
> $$P(X=x)=f(x)= \begin{cases}p & \text { if } x=1 \\ 1-p & \text { if } x=0\end{cases}$$
> $p=\text{probability of success}$
> 
> $\mu=E[X]=1 \cdot p+0 \cdot(1-p)=p$
> $\sigma^{2}=\operatorname{var}(X)=p(1-p)$
