# Uniform Density Function


> [!math]
> $f(x)=\left\{\begin{array}{cl} \frac{1}{\beta-\alpha} & \alpha \leq x \leq \beta \\ 0 & \text{ otherwise } \end{array}\right.$
> $a=\text{min}(x_i)$
> $\beta=\text{max}(x_i)$
