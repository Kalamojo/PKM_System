# Training Set

> [!note]
> The set of features used to train a [[Supervised Learning]] model in order to make predictions.

> [!math]
> $$S = \{(x_i, y_i)\}^{m}_{i=1}$$
> - $S=$ Training Set
> - $x_i=$ the $i^{ith}$ input
> - $y_i=$ the $i^{ith}$ output
> - $m=$ number of [[Data]] points
