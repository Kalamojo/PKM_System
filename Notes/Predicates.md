# Predicates


```ad-note
A pradicate is a proposition with (undefined) variables.
```

```ad-example
- $Q(x) = \text{x is a positive number}$
	- Q(x) is a predicate
- $P(x,y) = \text{x is larger than y}$
- $Q(x,y,z) = z > x + y$
```
