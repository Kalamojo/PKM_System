# Univariate Analysis

```ad-note
Univariate analysis focuses on a single attribute at a time. The data matrix $D$ is an n x 1 matrix, where $X$ is the numeric attribute of interest.
```

```ad-math
$$D = \begin{pmatrix} \underline{X}  \\ x_1 \\ x_2 \\ : \\ x_n\end{pmatrix}$$
```
