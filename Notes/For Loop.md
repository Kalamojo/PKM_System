# For Loop

> [!note]
> Loops repeat a certain number of times.

> [!example]
> ```cpp
> #include <iostream>
> 
> int main() {
>   int times = 10;
> 
>   for (int i = 0; i < times; i++) {
> 	  std::cout << "Dogs" << std::endl;
>   }
> }
