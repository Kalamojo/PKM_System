# Multi-dimensional array

Arrays but with extra dimensions to store values. In a two-dimensional array, the first number represents the rows while the second represents columns.

> [!example]
> 
| -   | -   | 15  | -   | -   |
| --- | --- | --- | --- | --- |
| -   | -   | -   | -   | 2   |
| -   | -   | -   | -   | -   | 
> ```cpp
 int A[3][5];
A[0][2] = 15;
A[1][4] = 2;
>```
