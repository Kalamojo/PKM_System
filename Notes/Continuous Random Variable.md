# Continuous Random Variable

> [!note] 
> A random variable can take on any value in R (any real number).

> [!math]
> $$A(v) = \begin{cases} x, & \text{for } 0 \le x \le 1 \\ 2-x, & \text{for } 1 < x \le 2 \\ 0, & \text{otherwise} \end{cases}$$
