# Shattered Set

>[!note]
> A [[Sets|Set]] is **shattered** by [[Hypothesis Class]] $\mathcal{H}$ if all possible ways of [[Classification|Classifying]] points in $S$ are achievable using hypotheses in $\mathcal{H}$.

> [!math]
> A [[Training Set]] is shattered by $\mathcal{H}$ if
> $$|\mathcal{H}[S]|=2^{|S|}$$
