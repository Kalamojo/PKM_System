# Correlation Test

> [!note]
> A statistical test in which the null hypothesis is that two attributes are not correlated. Uses the size of the attributes and the Pearson Correlation value.

> [!math]
> $$t = \frac{r \sqrt{n-2}}{\sqrt{1-r^2}}$$
>
> $\text{degrees of freedom} = n - 2$
