# Rasterization

>[!note]
> The process of taking an image described by [[Vector Graphics]] and converting it into a [[Data Matrix|grid]] of pixels.

> [!example]
> Raster graphics file formats:
> - JPEG
> - PNG
> - GIF
> - WebP
> - MPEG4
